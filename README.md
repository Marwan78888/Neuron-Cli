# 🧠 Neuron

**Your calm, local-first digital companion for serious engineering and polish.**

Neuron isn't just another terminal tool. It’s a teammate that lives in your codebase, understands your style, and helps you ship beautiful products with ease.

---

### ✨ Why you'll love Neuron

- **🤝 Acts like a teammate**: It doesn't just run commands; it plans, explores, and learns who you are.
- **🎨 Design-obsessed**: Built for engineers who care about how things look. It’s a pro at CSS, animations, and high-end layouts.
- **🏠 Local & Private**: Your data stays with you. It’s local-first and works with any AI provider you prefer.
- **🧬 Evolves with you**: Through the **Bootstrap Onboarding**, it learns your name, your vibe, and how you like to work.

---

### 🚀 Get Started in Seconds

Installing Neuron is as simple as running one command:

```bash
curl -fsSL https://raw.githubusercontent.com/Marwan78888/Neuron-Cli/main/install.sh | bash
```

Then, just start it in any project:

```bash
neuron start
```

---

### 🌟 What makes Neuron special?

#### 🙋‍♂️ Human-Agent Onboarding
When you first start Neuron in a new workspace, it wakes up and asks: *"Who am I? And who are you?"* 
Through a simple `BOOTSTRAP.md` process, it establishes an identity, chooses an emoji, and learns your preferences. It’s not just a script; it’s a presence.

#### 🕹️ The Terminal, Reimagined
Neuron features a stunning Terminal UI (TUI) that shows you exactly what it's thinking. You’ll see real-time progress like:
- `→ Refining the design...`
- `→ Exploring the codebase...`
- `→ Deploying to production...`

#### 🛠️ Professional Grade Power
- **Multi-step Execution**: It doesn't just edit one file; it solves whole problems.
- **MCP Support**: Connect it to any external tool or service seamlessly.
- **Cross-Provider**: Use Claude, GPT, or local models — you're in control.
- **Subagents**: Specialized agents can work in parallel on complex tasks.

### 🛠️ Developer Commands

If you're a power user, Neuron exposes a full CLI for every workflow:

```bash
neuron start     # Open the beautiful TUI
neuron run       # Run a task and finish
neuron mcp       # Manage tool connections
neuron skill     # Import or list skills
neuron github    # Supercharge your PR workflow
```
